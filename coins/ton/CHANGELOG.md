# Change Log

All notable changes to this project will be documented in this file.

# [0.0.5](https://github.com/okx/go-wallet-sdk) (2025-01-09)

### New features

-   **ton-sdk:** Add external signing support with `BuildMessageUnsigned` and `BuildMessageWithSignature` methods for V3, V4R2, and V5R1 wallets
-   **ton-sdk:** Add `Wallet.BuildMessageUnsigned()` and `Wallet.BuildMessageWithSignature()` for two-step transaction signing
-   **ton-sdk:** Update `RegularBuilder` interface with new unsigned/signature methods
